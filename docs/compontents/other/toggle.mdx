# Toggle

The `<Toggle/>` is a simple wrapper for a `<Checkbox/>` component to show it as a toggle.

## Import 

```jsx
import { Toggle } from "matro-ui";
```
- `Toggle`: Toggle input

 ## Usage

 ```jsx live 
  <Toggle>Toggle</Toggle>
```


 ## Toggle sizes

 Pass the <code>font-size</code> to change the size of the <I>Toggle</I>. Values can be either <code>px</code>, <code>em</code>, <code>rem</code> or <code>%</code>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
    <Toggle style={{fontSize: "14px"}} checked>
      Toggle
    </Toggle>
    <Toggle style={{fontSize: "16px"}} checked>
      Toggle
    </Toggle>
    <Toggle style={{fontSize: "1.2rem"}} checked>
      Toggle
    </Toggle>
  </Stack>
```

 ## Toggle with custom color

 You can override the color scheme of the <I>Toggle</I> to any color by providing the css variable <code>--bg</code>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
    <Toggle style={{"--bg": "red"}} checked>
      Toggle
    </Toggle>
    <Toggle style={{"--bg": "#48b07d"}} checked>
      Toggle
    </Toggle>
    <Toggle style={{"--bg": "#4848d7"}} checked>
      Toggle
    </Toggle>
 </Stack>
```
