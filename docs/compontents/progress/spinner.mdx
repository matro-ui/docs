 ## Usage

 ```jsx live 
  <Spinner />
```

## Spinner Size

To change the size of `<Spinner/>`, simply set the `size` prop (val.<I>"XS"</I> | <I>"S"</I> | <I>"M"</I> | <I>"L"</I> | <I>"XL"</I>):

 ```jsx live 
  <Spinner size="XL" />
```

## Color Scheme
 
To change the color of `<Spinner/>`, set the `colorScheme` prop (val. <I>"red"</I> | <I>"orange"</I> | <I>"yellow"</I> | <I>"green"</I> | <I>"cyangreen"</I> | <I>"cyanblue"</I> | <I>"blue"</I> | <I>"purple"</I> | <I>"pink"</I>):

 ```jsx live 
<Stack>
  <Spinner size="M" colorScheme="pink" />
  <Spinner size="M" colorScheme="orange" />
  <Spinner size="M" colorScheme="cyanblue" />
</Stack>
```

## Conditional Rendering

If you need to hide the `<Spinner/>`, you can set the `hidden` prop to <I>true</I>, or use a ternary JSX statement:

```jsx live 
function demo() {
  const [visible, setVisible] = React.useState(true);
  return (
    <Stack column>


    <LabelText>Example of hiding with propㅤ<I>"hidden"</I></LabelText>
    <Spinner hidden={!visible} />

    <LabelText>Example of conditional rendering</LabelText>
    {visible ? <Spinner /> : ""}

      
      <Button 
        onClick={() => setVisible(!visible)} 
        rightIcon="⚙️">
        toggle visibility
      </Button>
    </Stack>
  );
}
```
