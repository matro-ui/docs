# Checkbox

The `<Checkbox/>` component allows to select or deselect options in a form. It provides a customizable checkbox input with various options.

## Import 

```jsx
import { Checkbox } from "matro-ui";
```

 ## Usage

 ```jsx live 
  <Checkbox>Checkbox</Checkbox>
```

 ## Disabled Checkbox

 ```jsx live 
 <Stack style={{gap: 20}}>
    <Checkbox disabled>
      Checkbox
    </Checkbox>
    <Checkbox disabled checked>
      Checkbox
    </Checkbox>
  </Stack>
```

 ## Checkbox sizes

 Pass the <code>font-size</code> to change the size of the Checkbox. Values can be either <code>px</code>, <code>em</code>, <code>rem</code> or <code>%</code>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
    <Checkbox style={{fontSize: "10px"}} checked>
      Checkbox
    </Checkbox>
    <Checkbox style={{fontSize: "14px"}} checked>
      Checkbox
    </Checkbox>
    <Checkbox style={{fontSize: "1.6rem"}} checked>
      Checkbox
    </Checkbox>
  </Stack>
```

 ## Checkbox with custom color

 You can override the color scheme of the <I>Checkbox</I> to any color by providing the css variable <code>--bg</code>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
    <Checkbox style={{"--bg": "red"}} checked>
      Checkbox
    </Checkbox>
    <Checkbox style={{"--bg": "#48b07d"}} checked>
      Checkbox
    </Checkbox>
    <Checkbox style={{"--bg": "#4848d7"}} checked>
      Checkbox
    </Checkbox>
 </Stack>
```

 ## Checkbox as toggle

 Pass the <code>toggle</code> prop to change the view of the Checkbox.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
    <Checkbox style={{fontSize: "10px"}} checked toggle>
      Checkbox
    </Checkbox>
    <Checkbox style={{fontSize: "14px"}} checked toggle>
      Checkbox
    </Checkbox>
    <Checkbox style={{fontSize: "1.6rem", "--bg": "#48b07d"}} checked toggle>
      Checkbox
    </Checkbox>
 </Stack>
```

 ## Toggle Component

 Alternatively, you can use the <code>`<Toggle/>`</code> component. It wraps <code>Checkbox</code> and pass all props to it.

 ```jsx live 
  <Toggle style={{fontSize: "1.1rem"}} checked toggle>
    Toggle
  </Toggle>
```
