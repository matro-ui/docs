# Checkbox

The `<Checkbox/>` component allows to select or deselect options in a form. It provides a customizable checkbox input with various options.


 ## Checkbox sizes

 Pass the <I>font-size</I> to change the size of the Checkbox. Values can be either <I>px</I>, <I>em</I>, <I>rem</I> or <I>%</I>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
   <Checkbox style={{fontSize: "14px"}} checked>
    Checkbox
  </Checkbox>
  <Checkbox style={{fontSize: "16px"}} checked>
    Checkbox
  </Checkbox>
  <Checkbox style={{fontSize: "1.2rem"}} checked>
    Checkbox
  </Checkbox>
  </Stack>
```

 ## Checkbox with custom color

 You can override the color scheme of the <I>Checkbox</I> to any color by providing the css variable <I>--bg</I>.

 ```jsx live 
 <Stack style={{gap: 20, alignItems: "flex-end"}}>
  <Checkbox style={{"--bg": "red"}} checked>
    Checkbox
  </Checkbox>
  <Checkbox style={{"--bg": "#48b048"}} checked>
    Checkbox
  </Checkbox>
  <Checkbox style={{"--bg": "#4848d7"}} checked>
    Checkbox
  </Checkbox>
  </Stack>
```

## Props

- `value`: (boolean) The value of the checkbox.
- `onChange`: (function) Callback function triggered when the checkbox value changes.
- `name`: (string) The name attribute of the checkbox.
- `checked`: (boolean) Indicates whether the checkbox is checked.
- `toggle`: (boolean) Indicates whether the checkbox is in toggle mode.
- `linear`: (boolean) Indicates whether the checkbox has a linear style.

## Example Usage

```jsx
import React from 'react';
import {Checkbox} from 'matro-iu';

function App() {
  const handleCheckboxChange = (isChecked) => {
    console.log('Checkbox checked state:', isChecked);
  };

  return (
    <div>
      <Checkbox
        value={true}
        onChange={handleCheckboxChange}
        name="exampleCheckbox"
        checked={false}
        toggle={false}
        linear={true}
      >
        Example Checkbox
      </Checkbox>
    </div>
  );
}
```
