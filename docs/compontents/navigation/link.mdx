# Link as button

The `<LinkButton/>` component render link as button that allows to use `<a>` tag without breaking <a href="https://html.spec.whatwg.org/multipage/text-level-semantics.html#the-a-element" target="_blank">W3C markup rules</a>.

import { LinkButton } from "matro-ui";

 ## Usage

 ```jsx live 
  <LinkButton 
    as={<a href="https://example.com" target="_blank"></a>}
  >
   Hi! I'm a Link myself üëãüèæ
  </LinkButton>
```

 ## react-router-dom

 You can also use it as a wrapper to a <I>react-router-dom</I> `<Link/>`, or other navigation component

 ```jsx live 
  <LinkButton as={<Link to="#"/>}>
   LinkButton as {`<Link/>`}
  </LinkButton>
```

  <LinkButton 
    as={<Link to="#"/>} 
    leftIcon={<IoLayersOutline color="darkturquoise"/>}

  </LinkButton>
